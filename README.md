## 윤동성의 포트폴리오

### 💻프로젝트 소개
리액트로 개발한 저의 포트폴리오 페이지입니다.

### 📒사용 언어
- React.js

### 📗사용 라이브러리
- router
- axios
- emailJs
- framer-motion
- Intersection Observer
- redux
- styled-components

### ✏️주요 기능
- 원페이지 스크롤
- 스크롤 애니메이션
- About : 스킬과 사용 경험 있는 라이브러리 및 언어, 툴 기재
- Project: 개발한 프로젝트 슬라이더 형식, 모달로 구현
- Contact: 깃허브, 블로그 링크와 이메일, 메세지 입력 기능 구현
- 반응형 페이지로 제작
  - laptop: 1400px
  - tablet: 1024px
  - mobile: 768px
